# Installation

Step 1 (Install Nodejs)
```
pkg install nodejs-lts
```
Step 2 (Clone or Download)
```
git clone https://github.com/jaleelx98/Rambo-Bot.git
```
Step 2 (Open Rambo-Bot Drectory)
```
cd Rambo-Bot
```
Step 3 (Unzip)
```
unzip node_modules.zip
```
Step 4 (Run Rambo-Bot)
```
node index.js
```
